## 使用说明

1. 将上述每个 `Prompt` 区块的内容直接粘贴到 GPT image2 的对话框中
2. 每个 Prompt 生成一张独立的架构图
3. 可根据需要调整颜色、布局、详细程度
4. 建议生成顺序：图1（总览）→ 图2（数据流）→ 图5（AI引擎）→ 图4（后端）→ 图3（前端）→ 图6（导出）→ 图7（部署）→ 图8（技术栈）

### 快速生成单张综合图

如果只需要一张图概括整个系统，使用**图 1 系统总览架构图**的 Prompt 即可。
