## 6. 同一段对话在各表中的形态对比

```
raw_chats (6条独立记录):
  #1001 | "你好，我想了解一下你们的课程"  ← 单条消息
  #1002 | "你好呀～欢迎咨询！..."         ← 单条消息
  #1003 | "基础班多少钱？"                ← 单条消息
  #1004 | "基础班现在活动价2980..."        ← 含真实价格
  #1005 | "有点贵，能优惠吗？"            ← 单条消息
  #1006 | "现在报名可以送..."             ← 含真实价格

staging_conversations (1条记录):
  #42 | 上面6条合成一个对话块 | 价格已脱敏 | 带分类/评分/Q&A

knowledge_chunks (1条记录):
  #128 | staging的cleaned_text + 1024维向量 + 关键词

knowledge_articles (1条记录):
  #35 | LLM从对话中提炼出: 场景+客户说法+推荐回复（最精炼）

custom_conversations:
  不从raw_chats产生，是人工手写的"模范对话"
```
